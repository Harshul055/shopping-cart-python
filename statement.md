Project Title:

Shopping Cart Billing System (Python CLI Application)


---

Problem Statement

The goal of this project is to create a command-line based shopping cart and billing system that allows a user to:

View available products with their price and stock

Add and remove items from a shopping cart

Validate quantities and ensure stock accuracy

Apply a coupon code for discount

Generate a detailed bill with item frequency

Save a receipt as a text file

Provide delivery estimation and order ID


This system imitates a simplified version of an e-commerce checkout flow using Python.


---

Objective

To implement a real-world billing system in Python

To understand and practice the use of:

Dictionaries

Lists

Loops

Conditionals

File handling

User input validation

Modular programming

Frequency tables using Counter




---

Scope of the Project

The project includes the following functional components:

Login system

Product listing (30 items)

Stock management

Shopping cart operations

Billing with coupon support

Receipt generation

Delivery estimation


The project does not include:

GUI interface

Database server

Online storage

Multi-user authentication



---

Tools & Technologies Used

Language: Python 3

Libraries:

random (order ID, delivery date)

datetime (delivery estimation)

collections.Counter (frequency table)

time (optional delays)




---

Expected Output

Smooth user interaction through the terminal

Correct cart operations

Accurate bill calculation

Auto-generated receipt.txt file

Display of order details and delivery date



---

Conclusion

This project demonstrates the practical use of Python for building a realistic store billing application.
It also strengthens concepts of:

Data structures

Input handling

File operations

Modular coding

Logical problem-solving


The system successfully models the essential workflow of a basic supermarket checkout system.
